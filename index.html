Sejarah Kami

Semua ini berawal dari tahun 2021, saat saya menduduki bangku kelas 4 SD saya sudah mulai tertarik dengan dunia perkodingan. Saat itu saya mulai dari hal terkecil yaitu membuat subtitle film detective Conan, Karena disitu ada beberapa struktur HTML saya pun bertanya-tanya "Apa itu HTML?" (Di dalam benak saya). Saya pun Mulai mempelajari bahasa pemograman HTML di Website w3school. 2 Tahun sudah berjalan saat saya sudah menduduki bangku kelas 6 SD saya sudah lumayan mahir dalam bahasa pemograman HTML dan CSS, saya berpikir mungkin ini sudah waktunya untuk membuat website. 3 bulan lamanya website saya semuanya Gagal, Tetapi kegagalan itu tidak pernah membuat saya menyerah. Saya terus berusaha sampai sekarang, dan akhirnya sudah jadi website yang sekarang kalian kunjungi. "CodeGenius" kami melayani info aplikasi untuk anda yang menginginkan coding, blog, hingga kami dapat mengarsipkan web anda
